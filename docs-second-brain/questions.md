# Questions (Inquisitive Product Lens)

Use these prompts to turn creative friction into build priorities.

## File-management workflow questions

1. Where does the user lose trust in file moves (before, during, or after move)?
2. Which transition is most fragile right now: `Sketches -> Active`, `Active -> Archive`, or `Archive -> Active`?
3. What is the minimum conflict dialog that prevents accidental data loss?
4. Which file errors are currently console-only and invisible to the user?
5. Which scan failures should block action vs warn and continue?

## Flow-state questions

1. Which action should always be one keystroke away in each state?
2. Where is there decision fatigue in sample browsing?
3. Which prompts create momentum vs interruption?
4. What does a 10-minute "creative warm start" loop look like in FlowForge?
5. Which feature most increases time-to-first-sound?

## Navigation questions

1. Can a first-time user find all core actions without opening Settings again?
2. Which screen has hidden actions that should be promoted to primary UI?
3. What should happen when there is no folder configured for a state?
4. Where should "Reveal in Finder" be universal?
5. Which workflow path currently requires too many clicks?

## Session inquiry template

- What broke flow today?
- What caused uncertainty?
- What should be automated?
- What should remain manual and intentional?
